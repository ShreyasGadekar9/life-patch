# 🌿 Life Patch

**Life Patch** enables users to describe their issue in just a few words (e.g., “back pain”, “feeling low”) and instantly receive concise, meaningful guidance. The app combines AI-powered insights with practical recommendations, creating a seamless and helpful user experience.

---

## 🚀 Features

* 🧠 Get instant solutions by typing short problems (e.g., "back pain", "feeling low")
* 💡 AI-based short and clear suggestions
* 🛍️ Product recommendations related to the problem
* ⚡ Fast and minimal user interface
* 📱 Built using Flutter for cross-platform support

## ▶️ How to Run the App

1. Clone the repository:

   ```
   git clone https://github.com/ShreyasGadekar9/life-patch.git
   ```

2. Navigate to the project:

   ```
   cd life-patch
   ```

3. Get dependencies:

   ```
   flutter pub get
   ```

4. Run the app:

   ```
   flutter run
   ```
